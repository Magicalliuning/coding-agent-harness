# Project Instructions

## Agent skills

### Issue tracker

Issues and external PRs are tracked in GitHub and triaged with the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default mattpocock/skills triage labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Use a multi-context layout: root `CONTEXT-MAP.md` points to context-specific `CONTEXT.md` files, with shared ADRs under `docs/adr/`. See `docs/agents/domain.md`.
